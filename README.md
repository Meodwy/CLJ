# CLJ — Gestão Clínica

Sistema de gestão para clínicas desenvolvido com Next.js 16 + Supabase.

## Stack

- **Frontend:** Next.js 16, TypeScript, Tailwind CSS v4, shadcn/ui (base-ui/react)
- **Backend:** Supabase (Auth, PostgreSQL, Realtime)
- **Validação:** Zod v4

## Funcionalidades

- Cadastro de pacientes
- Agendamento de consultas
- Receituário (prescrições)
- Farmácia de manipulação
- Controle de estoque (FEFO)
- Financeiro (vendas, despesas)
- Relatórios

## Ambiente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).
